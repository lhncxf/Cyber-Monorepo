# Cyber — Web

Vue 3.5 + Vite 7 + Element Plus 前端管理后台 SPA。基于数据库驱动的动态路由与按钮级权限控制，内置完整的商品、订单、用户、部门、角色、菜单管理模块。

## 环境要求

- Node.js `>= 20.19` 或 `>= 22.12`
- pnpm `>= 10`

## 快速开始

### 1. 安装依赖

```bash
pnpm install
```

### 2. 启动开发服务器

```bash
pnpm dev
```

浏览器访问：http://localhost:5577

> 需先启动 `server` 后端服务，参见 [server 项目](../Cyber-Server/README.md)。

## 功能清单

| 模块 | 功能 |
|------|------|
| **登录** | 账号密码登录、SVG 图形验证码（JWT 无状态）、猫头鹰跟随动画 |
| **首页** | 角色差异化仪表盘（Admin / Editor 不同视图） |
| **系统管理 › 用户管理** | 用户列表分页查询、新增/修改/删除用户、重置密码、状态启禁用、按钮级权限控制 |
| **系统管理 › 角色管理** | 角色列表分页查询、新增/修改/删除角色、菜单权限勾选分配 |
| **系统管理 › 菜单管理** | 树形菜单列表、新增/修改/删除菜单（目录/页面/按钮三类型） |
| **系统管理 › 部门管理** | 树形部门列表、新增/修改/删除部门 |
| **商品管理 › 商品列表** | 商品分页查询、新增/修改/删除商品、SKU 多规格管理、分类级联选择 |
| **商品管理 › 分类管理** | 两级商品分类树、新增/修改/删除分类 |
| **商品管理 › 运费模板** | 运费模板分页查询、新增/修改/删除、区域附加运费与不发货地区配置 |
| **商品管理 › 订单管理** | 订单列表分页查询、查看订单详情、发货（填写物流单号与承运商） |
| **售后管理 › 退款申请** | 退款申请列表分页查询、审核通过 / 拒绝（批准后自动回滚库存） |
| **移动端管理 › 移动端用户** | 移动端用户列表分页查询、新增/修改/删除用户、重置密码、状态启禁用、查看收货地址 |
| **动态路由** | 路由由数据库驱动，登录后按角色动态加载，注销时自动清除 |
| **权限控制** | 按钮级权限指令 `v-permission`，不同角色显示不同操作按钮 |

## 命令

```bash
pnpm dev            # 开发服务器（端口 5577）
pnpm build          # 生产环境构建（输出到 dist/）
pnpm build:staging  # 预发布环境构建
pnpm lint           # ESLint 检查并自动修复
pnpm test           # 单元测试（Vitest）
```
